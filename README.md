# 📊Analyse des ventes d'une librairie avec Python
## 🎯Objectif
La Page est une librairie qui a plusieurs points de vente depuis 2 ans Lapage a ouvert un site de vente en ligne.
Le but du projet est d'analyser les ventes en ligne de la librairie La Page afin de les comparer aux ventes des librairies et de créer de nouvelles offres, adapter les prix et cibler une clientèle.  
Pour faire cette analyse je me suis appuyée sur 3 fichiers
- **customers** qui contient les données client (identifiant, genre et année de naissance)
- **products** qui contient les informations sur les livres (identifiant, prix et catégorie)
- **transactions** qui contient les informations sur les transactions (identifiant du livre, identifiant du client, date d'achat et le numéro de session lors de l'achat)

## 🛠️Outils
- Python
  - **Pandas** : Analyse et exploration des données
  - **Seaborn**, **Plotly** et **Matplotlib** : Visualisation des données
  - **Scipy** : Statistiques
## 🚀Résultats
- Les livres de la catégorie 0 sont moins chers et se vendent le plus
- Les livres de la catégorie 1 sont ceux qui génèrent le plus de chiffre d'affaire
- Les livres de la catégorie 2 sont les plus chers, les moins acheté et les plus rentables
